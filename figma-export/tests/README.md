# Skills Workflow Demo Screens Import

This package imports the current demo flow screenshots into a Figma page named `tests`.

## Screens

- `01-request.png`
- `02-budget.png`
- `03-approval.png`
- `04-project.png`
- `05-tasks.png`
- `06-resources.png`
- `07-execution.png`
- `08-proofing.png`
- `09-profitability.png`

## Use In Figma

1. Open the target Figma file.
2. Go to `Plugins` → `Development` → `Import plugin from manifest...`.
3. Select `figma-export/tests/manifest.json`.
4. Run `Skills Workflow Demo Screens Import`.

The plugin creates or reuses a page named `tests`, then adds the nine screens in a 3 by 3 layout.
